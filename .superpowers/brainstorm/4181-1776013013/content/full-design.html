Design Notes

X's & O's Hero Strip

Replaces the hero image. Pure CSS — no image file needed. A repeating X O pattern at low opacity (7%) in gold provides atmospheric depth. A subtle gold gradient line runs along the bottom edge. The tagline sits on top in secondary text.

① Scout — 4+3 Card Grid

7 trait categories shown as tap targets. Top row: 4 compact cards. Bottom row: 3 slightly wider cards. No lone card.

  • Tapping a card expands it inline below the grid — no modal, no nav
  • Active cards: green border + count badge
  • Desktop (≥768px): single 7-card row
  • Previously selected traits shown as gold chips inside expanded panel

② Bias Slider

Existing 7-position run/pass slider, unchanged visually. Anchored with numbered label so new users know it's Step 2.

③ Build CTA

Full-width gold gradient button. Enabled at ≥2 traits. "Save Opponent Profile" appears as a ghost button below it at ≥2 traits.

First-Time Onboarding Modal

Centered overlay. Triggers once via localStorage flag on first visit. Shows:

  • 3 numbered steps: Scout → Bias → Build
  • "You'll get" payoff pills: formations, coverages, blitz %, call sheets
  • Gold CTA: "Let's Build a Game Plan →"

Never shows again after dismissal. Toggle above to preview it.

Responsive / Desktop

  • 720px max-width container, centered (already in place)
  • Card grid: 4+3 on mobile → 7-across on desktop (≥768px)
  • Modal: centered, max-width 340px, works on any screen size
  • Bottom nav: stays fixed, same on desktop
